# dannecker-ITDEV117-Individual-Project
Completed project of an ordering app for a fictional tea house for ITDEV117 at MATC.

Application Name: Lucky Tea House Order App

Function: The Lucky Tea House Order App allows for contactless ordering to one's table within the Lucky Tea House, reducing queueing and allowing for staff to focus on preparing tea and other work beyond the cash register. The app is designed for in-house tablets which are available at the table. Additionally, the app has a timer feature which allows customers to steep their tea for several pre-set times which correlate to the types of tea offered to prevent customers from over or understeeping their tea.

Audience: Lucky Tea House customers

Key Features: The app allows for tea customizations, payment, customer loyalty sign-up, and has a timer to help customers steep their tea for the recommended amount of time.

How it works: As an in-house order application, all functions are designed to also terminate at the start, resetting the app to prompt new orders for continuing customers and ensuring it is ready for new customers. From the main menu, customers have the option to set a timer to steep their tea (in the event that they are re-steeping their tea after it has sat for awhile) or to view the menu to order. From the menu, customers are prompted to select a tea category (i.g. green or black) then a specific tea type in that category at which point they can customize the size and whether they want any additions like milk or sugar. At checkout, the customer has the option to log-in or sign-up for the rewards program. The customer can pay via credit or debit card on the app which will provide them with an order number before prompting them to start a timer or return to the main menu.
